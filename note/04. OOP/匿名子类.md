# 匿名子类

```java
public class Test {

    public static void main(String[] args) {
        A extendsA = new A() {};
        B b = new B();
        // new A() {} 实际上是 创建B类并实例化的简写

        X extendsX = new X(1) {};
        Y y = new Y(1);
        // new X(int x) {} 是 创建Y类并实例化的简写
    }

}

class A {
    public A() {}
}

class B extends A {
    public B() {}
}

class X {
    public X(int x) {}
}

class Y extends X {
    public Y(int x) {
        super(x);
    }
}
```
